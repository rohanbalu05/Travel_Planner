                         MANUAL SETUP (IF NEEDED)
================================================================================

If the batch files don't work, follow these steps:

1. Open PowerShell or Command Prompt
2. Navigate to this folder
3. Run these commands:

   python -m venv venv_nt
   venv_nt\Scripts\activate
   pip install -r requirements.txt
   python app_main.py

4. Open browser: http://127.0.0.1:5000

If you want real AI-generated itineraries:

1. Get free API key from: https://console.groq.com
2. Set it before running:

   PowerShell: $env:GROQ_API_KEY="your-key-here"
   CMD:        set GROQ_API_KEY=your-key-here

================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "Python is not recognized"
Solution: Make sure Python is installed and in PATH

Problem: Port 5000 already in use
Solution: Close other apps using port 5000

Problem: Dependencies won't install
Solution: Run PowerShell as Administrator and try again

Problem: Virtual environment won't activate
Solution: Use full path: venv_nt\Scripts\activate.bat
